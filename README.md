🚀 AllySense AI Analyzer
An advanced AI-powered Sentiment Analysis web application built using Natural Language Processing (NLP) and Machine Learning (ML).

AllySense AI Analyzer intelligently analyzes user reviews and predicts whether the sentiment is Positive or Negative with confidence scores and real-time analytics.

📌 Project Overview
In modern platforms like Amazon, Flipkart, Netflix, Swiggy, Instagram, and YouTube, millions of users post reviews and comments daily.

Companies use AI-based sentiment analysis systems to:

Understand customer satisfaction
Detect negative feedback
Improve products and services
Analyze public opinion
Track customer emotions
Generate business insights
AllySense AI Analyzer works on the same core idea.

This project uses:

NLP (Natural Language Processing)
Machine Learning Algorithms
Flask Web Framework
SQLite Database
Real-time Dashboard Analytics
Interactive UI Animations
The AI model analyzes text reviews and predicts:

✅ Positive Review ❌ Negative Review

along with prediction confidence percentage.

🧠 Core Concept
The project is based on Natural Language Processing (NLP).

NLP is a branch of Artificial Intelligence that enables computers to understand and process human language.

Example:

User Review	Prediction
"This product is amazing"	Positive
"Worst experience ever"	Negative
"Delivery was very bad"	Negative
"Excellent quality"	Positive
The AI model converts text into numerical vectors and predicts sentiment using Machine Learning.
