# AI Sentiment Analyzer 🚀

Professional NLP-based sentiment analysis web application using:

- Flask
- NLP
- Machine Learning
- Scikit-learn
- HTML/CSS/JS

## Features

- Positive / Negative / Neutral Prediction
- Confidence Score
- Dashboard Analytics
- Glassmorphism UI
- Machine Learning Model
- Real-Time Prediction

## Run Project

pip install -r requirements.txt

python train_model.py

python app.py
